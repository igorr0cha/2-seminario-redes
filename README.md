# 2-seminario-redes

Projeto para o 2º seminário de Redes — materiais, experimentos e códigos relacionados às atividades do seminário.

## Sobre
Este repositório reúne o material do 2º seminário de Redes: anotações, slides, códigos e relatórios. O objetivo é centralizar recursos e facilitar reprodução dos experimentos apresentados no seminário.

## Estrutura sugerida do repositório
- html com slide/conteúdo da apresentação
- README.md — este arquivo explicativo
